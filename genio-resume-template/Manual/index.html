OnePage Resume & Personal Portfolio Template

Thank you for purchasing my item. If you have any questions that are beyond the scope of this help file, please feel free to email us via our profile page contact form. Thanks so much!

Overview

Genio is a single page resume & personal portfolio template built with Flatron Bootstrap 3 Premium Theme. It is suitable for designers, developers & freelancers. Genio has a super clean, minimal & professional looking that anyone seeking for simplicity needs. It is packed with working Ajax/PHP hire & contact forms and a lot of other features which means it is ready for direct use.

Files & Folders




  1. Flatron Folder : Contains Flatron Bootstrap 3 Premium Theme main files and its manual.
  2. Manual Folder : Contains this documentation files.
  3. PSD Folder : Contains home page PSD file.

  4. CSS Folder : Contains all the stylesheets needed for coloring, typography and responsive layout.
  5. Fonts Folder : All your fonts should go here. By default it contains the fontawesome font icon.
  6. Inc Folder : This folder contains all PHP files. Files necessary for contact forms to work correctly.
  7. JS Folder : Contains all JavaScript files, plugins and libraries necessary for the item functionality.
  8. Media Folder : Contains Images Folder which contains all images used in the template.

CSS Part

Overview

Genio CSS files are very well organized to make it easy for you to find/edit/add/remove what you want. At the top of each file (main files only) you will find a detailed table of the whole file contents from the first line to the last just use Ctrl/f to navigate between different sections easily.

General

This template by default uses 6 CSS files for the sake of the development these files are splitted as you see but if you are going to use this template for production purpose it's recommended to combine them all in one file to decrease page loading time and here is how they are implemented in the <head> tag at the top of the document: 

<!-- ========================================= -->
<!-- $STYLESHEETS                              -->
<!-- ========================================= -->

<link rel="stylesheet" href="css/bootstrap.min.css">
<link rel="stylesheet" href="css/font-awesome.min.css">
<link rel="stylesheet" href="css/animate.min.css">
<link rel="stylesheet" href="css/magnific-popup.css">
<link rel="stylesheet" href="css/base.css">
<link rel="stylesheet" href="css/style.css">
<link rel="stylesheet" href="css/skins/yellow.css">

Also you will find a css stylesheet used to implement open sans google font as following: 

<!-- ========================================= -->
<!-- $GOOGLE FONTS                             -->
<!-- ========================================= -->

<link href='http://fonts.googleapis.com/css?family=Open+Sans:400,300,600,700,800' rel='stylesheet' type='text/css'>

And now it is time to know what each file is necessary for:

bootstrap.min.css
Boostrap framework file necessary for this template to look and function the way you see it.

font-awesome.min.css
Font icon CSS file containing all the classes you use in the HTML and the main styles for it.

animate.min.css
CSS animation library used for revealing content while scrolling with beautiful animation.

magnific-popup.css
Stylesheet for the portfolio lightbox plugin layout and general styles.

base.css
basic CSS styles for all template pages.

style.css
Main template CSS file needed for providing the layout and visual hierarchy as you see it.

[skin-color].css
Containing all template colors.

Details


style.css

This file is the main template file. You will find a table of contents at the top and we are going in a quick overview of the function of each section:

$TYPOGRAPHY
It contains the font sizes, families and styles and there responsive values that makes them adapt according to the device whether it is a desktop, tablet, phablet or phone. it also contains some helper classes that you can use across all the template like .em for italic text for more info open the file, navigate to the typography section and there you will find all the classes.

$HELPER CLASSES
This section contains all the necessary helper classes needed for the layout and so on like display classes need for setting element display property for example if you need to set an element display property to table just us .table class, there are also classes for visibility and block aligning besides default bootstrap classes to know it just open the file, navigate to helper classes section and start digging.

$SPACING
This section contains the necessary classes for the vertical & horizontal whitespace between elements and their different values on different devices, for example if you want to set the vertical white space on an element to 40px use the class .xlg-padder, for a complete list of classes just navigate to the spacing section and start digging.

$MODULES
This section contains the styles for the repeated elements like main title, secondary titles, line titles, rounded boxes ... etc, for more information just navigate to the section and start digging.

other sections
The remaining is the styles for each section like home, navigation, contact ... etc


skin.css

This file contains all template colors in one place to make changing color scheme to your desired colors is so easy task. This file is organized according to template colors and CSS properties for example if you want to edit the brown color you also need to define what CSS property you need to change color for, suppose it's background property, so here what you should do, navigate to brown color section find background property section and change the color from brown to whatever you want.so easy. 

find color name > find property name > change color

if you want to add color to an element here what you should do: 

find color name > find property name > concatenate name

Check for .hidden class > load specified number of items > get data-bg-src > set background > show

PHP Part

The only part using PHP in this template is the contact forms ( hire and message forms ) if you take a look at the inc folder you will find 4 files the only one that you need to change is config.php which requires providing contact and hire emails, you want to receive emails at. Also you should note that you can provide more that one email for each form type.

Requirements


  • PHP Version 5.x or higher.
  • PHP mail function enabled.

Form Installation


  • Open up inc/contact/config.php with your text editor and fill in your emails.
  • For single email, it should look like this:

    • $contact_emails = 'contact_email@example.com';
$hire_me_emails = 'hire_email@example.com';

  • For multi emails, it should look like this:

    • $contact_emails = 'contact_email@example.com, another_contact@example.com';
$hire_me_emails = 'hire_email@example.com, another_hire@example.com';

how to go online ?

01. After unzip the download pack, you'll found a Template Folder with all the files.
02. You can view this Template in any browser, you can display or edit the Template without an internet connection.
03. The only section that will not work is the Contact Section which contains PHP code and requires a server enviroment for send messages.
04. Now open your FTP Client (like Filezilla) and upload the content of the Template on your server root.
05. Once the files are done uploading go to www.yourdomainname.com/index.html
